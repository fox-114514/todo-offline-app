### 好友添加缺少审核机制

- 状态：已解决
- 优先级：高
- 影响范围：后端 / 安卓端 / 好友圈
- 是否需要重新打包 APK：是
- 是否需要重新部署后端：是
- 是否需要推送 GitHub：是

#### 现象

旧版本中，别人输入好友 ID 后会直接加入好友圈，没有自我介绍和被添加者审核流程，并且关系是单向的。

#### 修复记录

- 新增好友申请流程：申请人输入好友 ID 和自我介绍。
- 被申请人可以在安卓端查看收到的申请，并选择通过或拒绝。
- 申请通过后双方互加好友，可以互相查看公开 idea。
- 删除好友时双方关系一起解除。
