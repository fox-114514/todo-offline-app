# 更新日志

所有重要更新都会记录在这里。日期使用北京时间。

## 未发布

### 新增

- 安卓端好友圈 ID 增加复制按钮：好友圈页和设置页都可以一键复制自己的好友圈 ID。
- 增加好友申请审核流程：输入好友 ID 时需要填写自我介绍，被申请人通过后双方互加好友。
- 增加好友管理功能：可以查看好友列表、删除好友、点击好友查看公开 idea。
- 增加 idea 详情页：我的 idea 列表和好友圈 feed 都可以点击进入详情，查看完整内容和全部评论。
- 好友圈 feed 增加“我的公开”，自己发布的公开 idea 也能在好友圈里看到。
- 新增好友相关 API：`/api/friends`、`/api/friends/requests/*`、`/api/friends/{circleId}/ideas`。

### 变更

- 安卓版本升级为 `0.2.0`，`versionCode` 升级为 `2`，保持原 debug 签名以支持覆盖安装。
- 旧的单向好友圈关系在后端迁移时会自动补齐为双方好友关系。
- 旧的 `/api/circles/join` 不再直接加好友，只作为兼容入口创建好友申请。

### 修复

- 修复输入好友 ID 会直接单向加入的问题。
- 修复自己公开的 idea 不会出现在好友圈 feed 的问题。
- 修复评论弹窗只展示部分评论的问题，详情页现在展示全部未删除评论。

### 文档

- 增加 AI 与语音功能待更新文档。
- 增加待解决问题清单，并记录本轮功能问题的修复状态。
- 更新 README 和 API 文档，补充好友申请、好友管理、idea 详情和版本升级说明。

## 2026-06-09

### 新增

- 增加 Idea 好友圈功能：每个用户自动生成唯一好友圈 ID。
- 支持通过好友圈 ID 加入别人的好友圈。
- idea 支持私密和好友圈公开两种可见性。
- 增加好友圈 feed，可以刷已加入好友圈里的公开 idea。
- 增加点赞、取消点赞和一级评论功能。
- 新增 `/api/ideas`、`/api/feed`、`/api/circles/*`、点赞和评论相关 API。

### 修复

- 修复旧 SQLite 数据库升级到 v2 时，`visibility` 字段索引创建顺序导致后端容器反复重启的问题。
- 为旧数据库迁移补充自动化测试，确保旧数据会自动补齐 `visibility = private` 和用户 `circleId`。

### 文档

- 更新 README 的后端服务器部署教程。
- 更新 API 文档，补充好友圈、feed、点赞、评论接口。
- 更新需求文档，记录 V2 Idea 好友圈规划。

## 2026-06-08

### 新增

- 初始实现离线优先 ToDo/Idea 应用。
- 后端提供注册、登录、任务/idea CRUD、提醒设置、同步 pull/push API。
